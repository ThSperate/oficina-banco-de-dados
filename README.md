# Projeto Banco de Dados - Oficina

## Descrição
Este projeto implementa o banco de dados para o contexto de uma oficina, contendo informações de clientes, veículos, serviços e funcionários.

## Estrutura
O banco possui as seguintes tabelas principais:
- **Clientes**: dados pessoais e contato.
- **Veículos**: informações sobre os veículos dos clientes.
- **Serviços**: registros de serviços realizados.
- **Funcionários**: informações sobre os funcionários da oficina.

## Como usar
1. Abra o MySQL ou outro SGBD compatível.
2. Execute o arquivo `script.sql` para criar o banco de dados e todas as tabelas.
3. (Opcional) Insira dados de teste com os comandos fornecidos no `script.sql`.
